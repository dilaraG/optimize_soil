# Каталог `data/`

Рабочие файлы приложения (не передаются через Git — хранятся локально у инженера).

| Путь / файл | Назначение |
|-------------|------------|
| `ККД_БД.xlsx` | База лаборатории ККД; при первом импорте создаётся `kkd_lab.sqlite` |
| `kkd_lab.sqlite` | Кэш ККД после импорта (ускоряет повторные сеансы) |
| `uploads/` | Копии загруженных через интерфейс CSV/Excel (опционально) |
| `snapshots/` | Снимки для сравнения J/БК (CSV), путь задаёт пользователь в разделе «Сравнение методов» |
| `lab/` | Пользовательские лабораторные выгрузки (по желанию) |

Папки `uploads/` и при необходимости корень `data/` создаются автоматически при запуске приложения.
